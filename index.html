재미있고 흥미롭게 표현하는 것을 좋아해요.
머리 속에 생각한 것이 제가 쓴 코드 몇 줄로
동작하고 표현되는 것이 매력있고 재미있어서
지금 배우는 것을 즐기면서 하고 있습니다!

코딩을 할 때 항상 사용자의 입장에서 생각하며
사용자가 웹을 사용할 때 어떻게 하면 재미있게
사용할 수 있을까 고려하면서 만들고 있어요.

재밌는 공간, 사용자와 서비스가 소통하는 공간
그런 공간을 만드는 사람이 제가 꿈꾸는 저의 모습입니다.

공간이라는 곳은 기반이 튼튼해야 하며
다른 요소간의 조화가 잘 이루어져야 합니다.

기본에 충실한

개발도 마찬가지로 변하지 않는 기본이 중요하여
기본개념을 꾸준히 공부하고 있습니다. 특히 순수
자바스크립트 문법을 중요하게 여겨 서적이나 공식
문서 등을 통해 지속적으로 개념을 익히고 있습니다.

소통할 줄 아는

기획과 디자인을 배운 경험과 지속적인 팀프로젝트를
토대로 제가 맡는 분야 이외의 직군을 역할과 니즈를
이해할 수 있게 되었고 그들과 원활하게 소통을 하며
업무를 하기 위한 준비를 지속적으로 하고 있습니다.

우선 저는요-

제가 잘하는 것은-

HTML5

웹표준을 준수해서 HTML 마크업을 작성합니다. 시멘틱 마크업을 짜고 반응형웹을 중점적으로 다루고 있습니다. 단순한 하나의 태그의 반복이 아닌 각 용도에 맞는 태그를 사용하고자 항상 노력하고 있습니다.

CSS3

웹을 기획하는 단계에서 생각했던 제가 원하는 스타일링을 구현합니다. 특히 반응형 웹을 만들기 위한 Media query에 신경을 쓰고 있으며 레이아웃을 짤 때 Flexible Box와 Grid를 이용해 구현을 하여 어느 사이즈의 웹에서 보더라도 편안하게 볼 수 있는 스타일링을 하고 있고 애니메이션에 자신있습니다.

JavaScript

가장 최우선적으로 공부하고 있는 부분입니다. 웹의 동적인 요소를 구현하는 게 너무 재미있어서 다방면으로 공부하고 있으며 최대한 모든 부분의 구현을 직접 바닐라 자바스크립트로 만들어보고 있습니다. let, const, arrow function등 ES6의 문법에 대한 공부도 놓치기 않고 최신 문법에 대해 알아가고 있습니다.

jQuery

jQuery를 통한 셀렉터 사용에 익숙합니다. 자바스크립트의 코드를 줄여서 사용할 수 있다는 점과 이전까지 jQuery를 사용한 웹이 많기에 기본적인 지식들을 습득하고 활용하고 있습니다. 하지만 기본적인 자바스크립트 문법에 우선순위를 두고 있어서 자주 쓰지는 않습니다.

React JS

리액트로 프로젝트를 진행하면서 많이 익숙해졌습니다. Virtual DOM과 State, Component의 활용에 대한 이해도가 있으며 Functional Component를 통한 리액트 훅 사용이 가능합니다. 또한 리액트 라우터 라이브러리로 SPA를 만들어보며 클라이언트 사이드에서 이뤄지는 라우팅을 구현해보았고 RESTful API와 연동하여 프로젝트를 제작해보았습니다.

Post CSS

CSS 표준 문법에 대한 공부를 계속 하면서 접하게 되었습니다. PostCss 플러그인에 대해 지속적으로 공부 중에 있습니다. 리액트와 병행으로 공부하여 css의 모듈화를 활용했었고 autoprefixer 플러그인을 사용하여 크로스브라우징을 좀 더 간편하게 구현하고 있습니다.
